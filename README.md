**WordPress VAPTBOX – Next-Gen Recon & Enumeration Tool
**
WordPress VAPTBOX is an automated WordPress reconnaissance and vulnerability enumeration tool designed for pentesters and security researchers. It performs:

Footprinting & Web Fingerprinting (dig, nslookup, ping, traceroute, WhatWeb)

Network Scanning (Nmap full and service scans)

WordPress File & Directory Checks (readme, wp-admin, common paths)

Directory & URL Enumeration (Gobuster, Dirsearch, curl checks)

REST API & XML-RPC Enumeration

WPScan Enumeration (users, plugins, themes, vulnerabilities)

Brute Force Attacks (username/password combos, XML-RPC)

SSL & HTTP Enumeration (Nmap SSL scripts, headers, robots.txt, sitemap, etc.)

It is fully non-interactive, supports custom wordlists, max threads, and WPScan API token integration, providing a complete WordPress reconnaissance workflow with clear, sectioned output.
